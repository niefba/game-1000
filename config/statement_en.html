The Goal

Make more than 10,000 points.

  Rules

Players take turns.
The player receives 6 dice

If the dice are used to score points, you can either score the points or replay the dice.
To replay the dice, you must keep at least one of them.
If a dice roll fails to score points, all accumulated points are lost.

  Example

Player receives 6 dice with values 1 4 1 1 3 6
Player keeps all 1 value dice for a score of 1000 points providing to the standard output one line with 1 1 1
Player receives 3 dice with values 2 5 6
The player passes for a score of 50 points and scores a total of 1000 + 50 = 1050 points providing to the standard output one line with pass"

  Note

Warning: one turn allows the dice to be re-rolled a maximum of 2 times.

  Game Input

The program must first read the initialization data from standard input. Then, provide to the standard output one line with "pass" or a list of kept dices
Input

Line 1: one string count, the number of rolled dices

Line 2: one string values, the values of rolled dices

Output
A single line containing "pass" or a list of kept dices
Constraints
The list of kept dices can't be empty.
